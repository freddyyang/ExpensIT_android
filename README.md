
ExpensIT_android is an android app developed for budgeting and financing, allowing users to keep track of expenses in daily
There are four main classes in this project: BilldbHelper.java: helper class for manipulating the database; Frm_Addbills.java: the interface for adding a new entry; Grid_bills.java: the interface of the bill chart; MainActivity.java: the main page (introduction page);
